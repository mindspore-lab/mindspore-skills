# MindSpore Environment Deployment

Deploy a complete MindSpore development environment with CANN support for Huawei Ascend NPU.

Load the `mindspore-env-deploy` skill and follow its workflow.

## What This Does

1. Set up package management environment (uv or conda)
2. Download and install CANN packages
3. Install MindSpore wheel package
4. Verify installation with runcheck

## Key Components

- Package manager: uv (recommended) or conda
- CANN toolkit: Ascend Computing Architecture
- MindSpore: Deep learning framework
- Verification: Built-in runcheck utility
